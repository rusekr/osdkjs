Information about this example

This example demonstrates how to use oSDK to can create a simple application that allows you to carry audio or video calls.


Some explanations to the tutorial

We used bootstrap to style this example and added some modifications to the CSS file styles/styles.css. Pay attention that page style does not have any relation to the functional part of the tutorial and oSDK and has only decovative function.

In addition we used jQuery, which is also not necessary for oSDK and used only to simplify the work with the page. In scripts/scripts.js file we also added several utility functions to work with page, which also have no relation to the functional part of the example.

Step 3: Listeners for oSDK events

Functions assigned as a listener to process oSDK events assosiated with the connection: connecting ; connected ; disconnecting ; disconnected ; connectionFailed , displays information into a text field about the coming event, and then in case of connection unlock the controls, in case of no connection disable controls.

The second listener function processes the event of new media session gotMediaSession , which occurs in case of incoming or outgoing call while in the variable mediaSession stores a pointer to the media session, and then the information about the call is displayed in the text box and there are three listeners for the session: started ; ended ; failed .

// Listener for oSDK event: // connecting, connected, disconnecting, disconnected and connectionFailed oSDK.on(['connecting', 'connected', 'disconnecting', 'disconnected', 'connectionFailed'], function (event) { // Displaying information about the coming event print('Application got oSDK status: ' + event.type); if(event.type == 'connected') { // If oSDK is connected // output of the id of authorized user print('Connected as user: ' + event.user.id); // Activation controls $('#btnConnect').attr('disabled', 'disabled'); $('#btnDisconnect').attr('disabled', false); $('#btnAudioCall').attr('disabled', false); $('#btnVideoCall').attr('disabled', false); $('#btnAnswerCall').attr('disabled', false); $('#btnEndCall').attr('disabled', false); } else { // If oSDK is disconnected // Deactivation controls $('#btnConnect').attr('disabled', false); $('#btnDisconnect').attr('disabled', 'disabled'); $('#btnAudioCall').attr('disabled', 'disabled'); $('#btnVideoCall').attr('disabled', 'disabled'); $('#btnAnswerCall').attr('disabled', 'disabled'); $('#btnEndCall').attr('disabled', 'disabled'); } }); // Handling new media session created by own call method or by other party oSDK.on('gotMediaSession', function (data) { // Save a media session in a variable mediaSession = data; // Message bout incoming call or starting outgoing call if (mediaSession.incoming) { print('Received ' + (mediaSession.isVideoCall ? 'video' : 'audio') + ' call from ' + mediaSession.from + '.'); } else { print((mediaSession.isVideoCall ? 'Video' : 'Audio') + '-calling to ' + mediaSession.to + '.'); } // Handling user interface on call started event mediaSession.on('started', function () { // Displaying call status in two cases if (mediaSession.incoming) { print('Call answered'); } else { print('Call answered by ' + mediaSession.to + ''); } // Binding opponent's audio or video stream to our displaying html element mediaSession.attachLocalStream(tagToOutgoingMediaStream); mediaSession.attachRemoteStream(tagToIncomingMediaStream); }); // Handling call ending event mediaSession.on('ended', function () { // Displaying fact that call ended print('Call ended'); // Simply unbinding anything from our displaying html element mediaSession.detachStream(tagToOutgoingMediaStream); mediaSession.detachStream(tagToIncomingMediaStream); }); // Handling call failing event mediaSession.on('failed', function (initiator, response, cause) { // Displaying fact that call ended print('Call failed on ' + initiator + ' side with error text: ' + cause); // Simply unbinding anything from our displaying html element mediaSession.detachStream(tagToOutgoingMediaStream); mediaSession.detachStream(tagToIncomingMediaStream); }); });

  • event handler started - displays information in the text box about the beginning of the call and attaches the incoming and outgoing media streams to the corresponding elements: video#tagToOutgoingMediaStream и video#tagToIncomingMediaStream;
  • event handler ended - displays information in the text box about the completion of the call and detaches from the elements corresponding media streams;
  • event handler failed - reports about error and detaches from the elements corresponding media streams, if any streams have been appointed;

Step 4: Processing pressing the buttons

When you press the button button#btnConnect oSDK initialization starts with the application ID, which should be first entered in the field input#applicationId, after that the process of connecting oSDK starts.

When you press the button button#btnDisconnect the process of disconnecting oSDK starts.

// Processing pressing button button#btnConnect $('#btnConnect').click(function() { // Initialization oSDK with the application ID taken from the field input#applicationId // the pointer to which is stored in variable appIdElement oSDK({appID: appIdElement.val()}); // Connecting oSDK oSDK.connect(); return false; }); // Processing pressing button button#btnDisconnect $('#btnDisconnect').click(function() { // Disconnecting oSDK oSDK.disconnect(); return false; });

The next four buttons manage call handling. Buttons button#btnAudioCall and button#btnVideoCall make call to subscriber. The caller ID will be taken from the field input#contactId for the call.

In the case of an incoming call,that is reported in the text box , by pressing the button button#btnAnswerCall this call will be accepted and a conversation will start.

Нажатие на кнопку button#btnEndCall приведет к завершению текущего звонка.

// Processing pressing audio call button $('#btnAudioCall').on('click', function () { // Get Contact ID var contact = $('#contactId').val(); if (contact) { // Audio call oSDK.audioCall(contact); } return false; }); // Processing pressing audio call button $('#btnVideoCall').on('click', function () { // Get Contact ID var contact = $('#contactId').val(); if (contact) { // Video call oSDK.videoCall(contact); } return false; }); // Processing pressing answer call button $('#btnAnswerCall').on('click', function () { // Receive call of the current media session mediaSession.answer(); return false; }); // Processing pressing end call button $('#btnEndCall').on('click', function () { // Ebd call of the current media session mediaSession.end(); return false; });
