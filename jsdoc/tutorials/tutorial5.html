Information about this example

This example will demonstrate the work of oSDK with presences: status change and signature change for current user.

To see how this example works, you need two browsers and two registered users who are in contact list of each other and signed on each other.


Some explanations to the tutorial

We used bootstrap to style this example and added some modifications to the CSS file styles/styles.css. Pay attention that page style does not have any relation to the functional part of the tutorial and oSDK and has only decovative function.

In addition we used jQuery, which is also not necessary for oSDK and used only to simplify the work with the page. In scripts/scripts.js file we also added several utility functions to work with page, which also have no relation to the functional part of the example.

// Initialization of the group of tabs, containing examples // tutorial, html, css, js initTabs(); // Decorations of blocks containing code examples // html, css, js initCodeBlocks(); // Saving pointer to the field input#applicationId var appIdElement = $('#applicationId'); // Function to display text messages to element textarea#logs function print(text) { // Adding text into an element $('#logs').html($('#logs').html() + (new Date()).toLocaleTimeString() + ': ' + text + '\n'); // Scrolling element to the bottom edge $('#logs').scrollTop($('#logs')[0].scrollHeight - $('#logs').height()); } // This function displays the contents of the contact list in the field select#contactsList function contactsList() { // Getting a contact list var contacts = oSDK.getContacts(); // Clear field select#contactsList $('#contactsList').empty(); // Displays a contact list if (contacts && contacts.length) { $.each(contacts, function(index, value) { var option = document.createElement('option'); option.setAttribute('value', value.id); var text = value.id + ' : ' + value.status + ' : ' + value.signature; option.innerHTML = text; document.getElementById('rosterList').appendChild(option); }); } }

Step 3: Listeners for oSDK events

Two functions are assinged in the example for oSDK event handling: the first of these processes oSDK events , associated with the installation of connection: connecting; connected; disconnecting; disconnected; connectionFailed.

The second listener function processes the event contactWasUpdated, which occurs when a contact from the contact list of the current user to change any of its data.

// Listener for oSDK event: // connecting, connected, disconnecting, disconnected and connectionFailed oSDK.on([ 'connecting', 'connected', 'disconnecting', 'disconnected', 'connectionFailed' ], function (event) { // Displaying information about the coming event print('Application got oSDK status: ' + event.type); if(event.type == 'connected') { // If oSDK is connected // output of the id of authorized user print('Connected as user: ' + event.user.id); // Activation controls $('#btnConnect').attr('disabled', 'disabled'); $('#btnDisconnect').attr('disabled', false); $('#btnSetStatusChat').attr('disabled', false); $('#btnSetStatusAway').attr('disabled', false); $('#btnSetStatusDND').attr('disabled', false); $('#btnSetStatusXA').attr('disabled', false); $('#btnSetSignature').attr('disabled', false); // Display the contact list of current user contactsList(); // Displays the status and signature of current user $('#currentStatus').val(oSDK.getClient().status); $('#currentSignature').val(oSDK.getClient().signature); } else { // If oSDK is disconnected $('#btnConnect').attr('disabled', false); $('#btnDisconnect').attr('disabled', 'disabled'); $('#btnSetStatusChat').attr('disabled', 'disabled'); $('#btnSetStatusAway').attr('disabled', 'disabled'); $('#btnSetStatusDND').attr('disabled', 'disabled'); $('#btnSetStatusXA').attr('disabled', 'disabled'); $('#btnSetSignature').attr('disabled', 'disabled'); } }); // ContactWasUpdated event occurs when          // someone from the contact list of current user          // connected, disconnected, changed status, or changed the signature oSDK.on('contactWasUpdated', function(response) { print('Contact ' + response.contact.id + ' was updated'); contactsList(); });

Step 4: Processing pressing the buttons

When you press the button button#btnConnect oSDK initialization starts with the application ID, which should be first entered in the field input#applicationId, after that the process of connecting oSDK starts.

When you press the button button#btnDisconnect the process of disconnecting oSDK starts.

// Processing pressing button button#btnConnect $('#btnConnect').click(function() { // Initialization oSDK with the application ID taken from the field input#applicationId // the pointer to which is stored in variable appIdElement oSDK({appID: appIdElement.val()}); // Connecting oSDK oSDK.connect(); return false; }); // Processing pressing button button#btnDisconnect $('#btnDisconnect').click(function() { // Disconnecting oSDK oSDK.disconnect(); return false; });

Pressing one of the four buttons: button#btnSetStatusChat; button#btnSetStatusAway; button#btnSetStatusDND; button#btnSetStatusXA will lead to a status change of the current user, and the status is defined by the button text content. After changing the status information about the changes will be displayed in the field input#currentStatus.

When you press button button#btnSetSignature handler function will take the signature from the field input#signature and set it for the currently logged in user, and the information about the signature change will be displayed in the field input#currentSignature.

If a user connects or disconnects, changes its status or signature, all other users who have him in the contact list and are subscribed to the notification with the user, receive notification of user data changes.

// Processing pressing one of the buttons for setting status $('#btnSetStatusChat,#btnSetStatusAway,#btnSetStatusDND,#btnSetStatusXA').click(function() { // Determining the status of contact var status = null; switch($.trim($(this).html()).toLowerCase()) { case 'chat' : status = 'chat'; break; case 'away' : status = 'away'; break; case 'do not disturb' : status = 'dnd'; break; case 'x-available' : status = 'xa'; break; } // Set status oSDK.setStatus(status, { onError: function() { // Error message print('Error!'); }, onSuccess: function() { // Display set status $('#currentStatus').val(oSDK.getClient().status); // Message about status change print('Status changed to: ' + oSDK.getClient().status); } }); return false; }); // A button to set your signature $('#btnSetSignature').click(function() { var signature = $('#signature').val(); // Set signature oSDK.setSignature(signature, { onError: function() { // Error message print('Error!'); }, onSuccess: function() { // Display set signature $('#currentSignature').val(oSDK.getClient().signature); // Clean the input signature line $('#signature').val(''); // Message about signature change print('Signature changed to: ' + oSDK.getClient().signature); } }); return false; });
